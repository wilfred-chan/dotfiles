# Arch Linux dotfiles

## Set-ups
### Browser
- qutebrowser
- chromium
### System Utilities
- doas (sudo is bloated and insecure!)
- pikaur
- brightnessctl
- pulseaudio
- acpi (battery info)
### Basic Utilities
- sway (wayland-native window manager)
- waybar (status bar for sway)
- foot (wayland-native terminal, my config disabled HiDPI detect)
- neovim
- git
### Personal Utilities
- 1password-cli
- s-tui
